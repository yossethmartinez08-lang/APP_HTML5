Aprendiendo HTML5

Esta página está dedicada a explicar qué es HTML5, su estructura, sus etiquetas y su importancia en el desarrollo web. El contenido está diseñado para facilitar el aprendizaje de los fundamentos de este lenguaje esencial.

Comenzar

¿Qué es HTML5?

HTML5 es la quinta versión del Lenguaje de Marcado de Hipertexto, el estándar que define la estructura y el contenido de las páginas web. A diferencia de versiones anteriores, incorpora nuevas características semánticas, soporte nativo para multimedia y mejoras en la accesibilidad, lo que lo convierte en la base fundamental para el desarrollo web moderno, permitiendo crear sitios más ricos, interactivos y compatibles con diversos dispositivos.

Estructura básica de HTML5

<!DOCTYPE html>
<html>
<head>
<title>Mi página</title>
</head>
<body>
</body>
</html>

La estructura de un documento HTML5 comienza con la declaración <!DOCTYPE html>, que indica al navegador el tipo de documento. La etiqueta <html> envuelve todo el contenido y se divide en dos secciones principales: <head> y <body>. El <head> contiene metadatos, el título y enlaces a recursos externos, mientras que el <body> alberga todo el contenido visible de la página, como textos, imágenes y otros elementos que interactúan con el usuario.

Etiquetas principales

h1 - h6

Las etiquetas <h1> a <h6> definen los encabezados o títulos de la página, donde <h1> representa el título más importante y <h6> el menos relevante. Se utilizan para estructurar jerárquicamente el contenido, facilitando tanto la lectura humana como la interpretación por parte de los motores de búsqueda.

p

La etiqueta <p> se utiliza para definir párrafos de texto. Es el elemento básico para agrupar contenido textual, permitiendo que los navegadores apliquen espaciado automático entre bloques de texto, lo que mejora la legibilidad y la organización de la información en la página.

img

La etiqueta <img> permite insertar imágenes en la página web. Es un elemento vacío que requiere atributos esenciales como src para indicar la ruta de la imagen y alt para proporcionar una descripción textual, fundamental para la accesibilidad y en caso de que la imagen no se cargue.

a

La etiqueta <a> crea enlaces o hipervínculos, permitiendo navegar entre páginas o secciones del mismo documento. El atributo href especifica el destino del enlace, y su contenido, que puede ser texto o una imagen, actúa como el elemento clicable para el usuario.

button

La etiqueta <button> define un botón clicable que puede ser utilizado para activar acciones, como enviar formularios o ejecutar scripts. Puede contener texto, imágenes u otros elementos HTML, y su comportamiento se puede personalizar mediante atributos o código JavaScript.

ul / ol / li

Las etiquetas <ul> y <ol> crean listas desordenadas (con viñetas) y ordenadas (numeradas), respectivamente. Dentro de ellas, la etiqueta <li> define cada elemento de la lista. Son esenciales para presentar información de forma estructurada y fácil de leer, como menús de navegación o listas de características.

HTML5 semántico

header

La etiqueta <header> representa el encabezado de una página o de una sección. Suele contener logotipos, títulos, menús de navegación u otros elementos introductorios, ayudando a definir claramente el inicio de un contenido y mejorando la estructura lógica del documento.

nav

La etiqueta <nav> se utiliza para agrupar enlaces de navegación, como menús principales o barras laterales. Al identificar explícitamente las secciones de navegación, facilita que los lectores de pantalla y otros dispositivos de asistencia interpreten y naveguen el sitio de manera más eficiente.

main

La etiqueta <main> delimita el contenido principal del documento, el cual es único y central para el propósito de la página. Debe excluir contenido que se repita en varias páginas, como encabezados o pies de página, sirviendo como punto de referencia clave para la estructura del sitio.

section

La etiqueta <section> agrupa contenido temáticamente relacionado, formando una sección lógica dentro del documento. Generalmente incluye un encabezado y se utiliza para dividir el contenido en partes significativas, lo que facilita tanto la organización del código como la comprensión de la estructura por parte de los navegadores.

article

La etiqueta <article> encierra contenido independiente y autónomo, que podría existir por sí mismo fuera del contexto de la página, como un artículo de blog, una noticia o un comentario. Su uso ayuda a estructurar la información de manera que sea fácilmente reutilizable y comprensible por agentes externos.

footer

La etiqueta <footer> define el pie de página de un documento o sección. Suele contener información como derechos de autor, enlaces a políticas de privacidad, datos de contacto o créditos, marcando el final del contenido y proporcionando contexto adicional al usuario.

Formularios en HTML5

Los formularios en HTML5 son herramientas fundamentales para recopilar información del usuario. Incluyen elementos como <form> para definir el formulario, <label> para etiquetar campos, <input> para diversos tipos de entrada (texto, correo, etc.), <textarea> para textos largos, <select> para listas desplegables y <button> para acciones como envío. HTML5 añadió nuevos tipos de entrada y atributos que mejoran la validación y la experiencia de usuario sin necesidad de JavaScript adicional.

Multimedia en HTML5

HTML5 introdujo soporte nativo para contenido multimedia, permitiendo integrar imágenes, audio y video sin depender de plugins externos. Esto no solo simplifica el desarrollo, sino que también mejora el rendimiento y la accesibilidad, ofreciendo una experiencia más fluida y consistente en diferentes dispositivos y navegadores.

Imágenes

La etiqueta <img> es la forma estándar de insertar imágenes. Además de los atributos obligatorios src y alt, se pueden usar atributos como width y height para controlar el tamaño. HTML5 también soporta formatos modernos y elementos como <picture> para ofrecer diferentes versiones de una imagen según el dispositivo.

Audio

La etiqueta <audio> permite incrustar contenido de sonido en la página. Soporta formatos como MP3, WAV y OGG, y ofrece atributos como controls para mostrar la interfaz de reproducción, autoplay para iniciar la reproducción automáticamente y loop para repetir el audio, facilitando la integración de música o efectos de sonido.

Video

La etiqueta <video> permite incrustar videos directamente en la web. Soporta formatos como MP4, WebM y OGG, y cuenta con atributos similares al audio, además de opciones para mostrar subtítulos mediante la etiqueta <track>. Esto permite crear reproductores de video personalizados y accesibles sin necesidad de software adicional.

Accesibilidad

El atributo alt en las imágenes es crucial para la accesibilidad, ya que proporciona una descripción textual que es leída por los lectores de pantalla para usuarios con discapacidad visual. Además, para audio y video, se deben proporcionar subtítulos, transcripciones y descripciones para asegurar que todo el contenido multimedia sea accesible al mayor número de personas posible.

Conclusión

HTML5 representa un avance significativo en el desarrollo web, estableciendo un estándar más potente, flexible y accesible. Al combinar una estructura semántica mejorada con soporte nativo para formularios interactivos y contenido multimedia, permite a los desarrolladores crear experiencias ricas y dinámicas que funcionan en una amplia variedad de dispositivos. Dominar HTML5 es esencial para cualquier profesional del área, ya que forma la base sobre la que se construyen las aplicaciones y sitios web modernos, influyendo directamente en la usabilidad, el rendimiento y la llegada a una audiencia global.
